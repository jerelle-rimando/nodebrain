# 🧠 NodeBrain — AI Agent Command Center

NodeBrain™ is a local-first (with potential for web deployment) system for building and running AI agents that actually do things.
Agents are created from chat and turned into persistent nodes in a visual graph. They can execute tasks on time, interact with external tools, and run structured workflows while you monitor everything in real time.

The interface combines three core systems:

Dashboard – create, control, and modify agents through chat


NodeGraph – visualize tasks, dependencies, and execution status


Credential Vault – securely connect APIs and integrations

Everything runs locally and you can choose your preferred model by putting your API key in the Credential Vault. 

NodeBrain is designed for solo developers and builders who want transparent, controllable AI workflows instead of black-box or fully autonomous assistants.

If you want an AI system that acts like a command center for automation, NodeBrain is built for that.
